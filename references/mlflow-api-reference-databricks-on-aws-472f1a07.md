The open-source MLflow REST API allows you to create, list, and get experiments and runs, and allows you to log parameters, metrics, and artifacts. The Databricks Runtime for Machine Learning provides a managed version of the MLflow server, which includes experiment tracking and the Model Registry.

For MLflow, there are two REST API reference guides:

*   [Databricks MLflow REST API 2.0 reference](https://docs.databricks.com/api/workspace/experiments)
*   [Open Source MLflow REST API reference](https://mlflow.org/docs/latest/rest-api.html)
