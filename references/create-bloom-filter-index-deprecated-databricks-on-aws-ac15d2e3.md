Bloom filter indexes are deprecated. Do not create new Bloom filter indexes. Use [predictive I/O](https://docs.databricks.com/aws/en/optimizations/predictive-io) or [liquid clustering](https://docs.databricks.com/aws/en/tables/clustering) instead. See [Bloom filter indexes (deprecated)](https://docs.databricks.com/aws/en/optimizations/bloom-filters) for details and migration guidance.
