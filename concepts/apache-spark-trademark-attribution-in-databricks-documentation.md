# Apache Spark Trademark Attribution in Databricks Documentation

**Apache Spark Trademark Attribution in Databricks Documentation** refers to the legal requirement for Databricks to display the appropriate trademark attribution for Apache Spark, Spark, and the Spark logo within its documentation and other materials. This attribution acknowledges that these marks are trademarks of the Apache Software Foundation (ASF).

## Overview

Databricks documentation materials include a copyright and trademark notice that attributes the Apache Spark, Spark, and Spark logo trademarks to the Apache Software Foundation. This attribution is standard across all Databricks documentation and is required to comply with the ASF's trademark policies.^[applicable-model-terms-databricks-on-aws.md]

## Standard Attribution Text

The standard attribution statement appears as follows:

> Apache, Apache Spark, Spark and the Spark logo are trademarks of the [Apache Software Foundation](https://www.apache.org/).^[applicable-model-terms-databricks-on-aws.md]

## Scope of Attribution

The attribution covers three distinct trademarked elements:^[applicable-model-terms-databricks-on-aws.md]

| Element | Description |
|---------|-------------|
| **Apache** | The foundation-level brand mark |
| **Apache Spark** | The full project name, which includes the foundation prefix |
| **Spark** | The shortened project name commonly used in documentation |
| **Spark logo** | The graphical logo associated with the Apache Spark project |

## Legal and Compliance Context

This attribution reflects Databricks' compliance with the Apache Software Foundation's trademark policies. As Databricks' platform is built around Apache Spark, proper attribution is necessary to:^[applicable-model-terms-databricks-on-aws.md]

- Distinguish Databricks' products from the Apache Spark open-source project
- Acknowledge the ASF's ownership of the trademarks
- Meet the licensing and branding requirements of the Apache License

## Placement in Documentation

The attribution appears in the footer or copyright section of Databricks documentation pages, typically alongside the general copyright notice that reads "© Databricks 2026. All rights reserved."^[applicable-model-terms-databricks-on-aws.md]
