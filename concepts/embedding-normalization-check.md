## Embedding Normalization Check

An **embedding normalization check** is a verification step to determine whether the vector output of an [embedding model](/concepts/text-embedding-models.md) is unit‑normalized, i.e., has a Euclidean norm (L2 magnitude) of approximately 1.0. Normalization is a common property of many embedding models because it simplifies downstream operations such as cosine similarity (which reduces to a simple dot product between normalized vectors). When using Databricks [Foundation Model APIs](/concepts/foundation-model-apis.md) or [External Models](/concepts/external-models.md) for embeddings, the user can verify whether the returned embeddings are normalized before feeding them into similarity search, clustering, or other vector‑based algorithms. ^[query-an-embedding-model-databricks-on-aws.md]

### Checking Normalization

Databricks provides the following Python function to check whether an embedding vector is normalized. The function computes the L2 norm of the vector and compares it to 1.0 within a tolerance (default `1e-3`). ^[query-an-embedding-model-databricks-on-aws.md]

```python
import numpy as np

def is_normalized(vector: list[float], tol: float = 1e-3) -> bool:
    magnitude = np.linalg.norm(vector)
    return abs(magnitude - 1) < tol
```

The function returns `True` if the vector’s magnitude is sufficiently close to 1.0; otherwise it returns `False`. The tolerance parameter `tol` accounts for floating‑point rounding errors that may cause the norm to deviate slightly from exactly 1.0.

### Usage Notes

- The function relies on NumPy for the norm calculation; `np.linalg.norm` computes the L2 (Euclidean) norm by default.
- A tolerance of `1e-3` is typically adequate for most embeddings, but users may adjust it based on the precision of their model or framework.
- Normalization status is **not** guaranteed to be stable across different models or even across different runs of the same model; it is good practice to verify programmatically when the model card or documentation is ambiguous.
