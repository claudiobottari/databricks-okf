# MLflow REST API

The **MLflow REST API** is the programmatic interface for interacting with the MLflow tracking server. It provides endpoints to create, list, and retrieve experiments and runs, and to log parameters, metrics, and artifacts.^[mlflow-api-reference-databricks-on-aws.md]

## Managed Version on Databricks

The Databricks Runtime for Machine Learning includes a managed version of the MLflow server that supports [MLflow Experiment Tracking](/concepts/mlflow-experiment-tracking.md) and the [MLflow Model Registry](/concepts/mlflow-model-registry.md).^[mlflow-api-reference-databricks-on-aws.md]

## API Reference Guides

Two REST API reference guides are available:

- **Databricks MLflow REST API 2.0 reference** — covers the managed service on Databricks.  
- **Open Source MLflow REST API reference** — covers the open‑source version of the API.^[mlflow-api-reference-databricks-on-aws.md]
