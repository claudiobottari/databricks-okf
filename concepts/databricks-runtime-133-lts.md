# Databricks Runtime 13.3 LTS

**Databricks Runtime 13.3 LTS** is a long-term support (LTS) version of the Databricks Runtime that serves as a stable foundation for production workloads. It is the minimum runtime version required for [Databricks Connect for Scala](/concepts/databricks-connect-for-scala.md) (version 13.3 LTS and above). ^[databricks-connect-for-scala-databricks-on-aws.md]

## Supported Features

Databricks Runtime 13.3 LTS is the baseline for [Databricks Connect](/concepts/databricks-connect.md), enabling developers to connect popular IDEs such as IntelliJ IDEA, notebook servers, and custom applications to Databricks compute clusters running this runtime version. For details on using Databricks Connect with this runtime, see [Databricks Connect for Scala](/concepts/databricks-connect-for-scala.md), [Databricks Connect for Python](/concepts/databricks-connect-for-python.md), and [Databricks Connect for R](/concepts/databricks-connect-for-r.md). ^[databricks-connect-for-scala-databricks-on-aws.md]
