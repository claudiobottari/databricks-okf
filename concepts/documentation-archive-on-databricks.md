## Purpose and structure

The documentation archive exists to preserve access to older versions of guides, tutorials, and reference materials that may still be relevant for users running legacy workflows or understanding the evolution of Databricks features. Common archived subjects include deprecated algorithms, older API references, and out‑of‑date configuration patterns for serverless or GPU compute environments. ^[deep-learning-using-tensorflow-with-horovodrunner-for-mnist-databricks-on-aws.md]
