# Databricks Utilities with Databricks Connect

**Databricks Utilities with Databricks Connect** refers to the ability to use Databricks Utilities (`dbutils`) when developing locally with [Databricks Connect for Python](/concepts/databricks-connect-for-python.md). The Databricks documentation provides a dedicated resource that explains how to use `dbutils` features within the Databricks Connect environment. ^[databricks-connect-for-python-databricks-on-aws.md]

The official documentation for Databricks Utilities with Databricks Connect for Python can be found at the following URL: https://docs.databricks.com/aws/en/dev-tools/databricks-connect/python/databricks-utilities. ^[databricks-connect-for-python-databricks-on-aws.md]
