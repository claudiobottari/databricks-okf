# Databricks Delta Lake Documentation

**Delta Lake** is an [open source storage layer](https://delta.io/) that brings reliability to data lakes. It provides ACID transactions, scalable metadata handling, and unifies streaming and batch data processing. Delta Lake runs on top of your existing data lake and is fully compatible with Apache Spark APIs. ^[delta-lake-api-reference-databricks-on-aws.md]

## API References

For detailed [Delta Lake](/concepts/delta-lake.md) API documentation for Scala, Java, and Python, see the [Delta Lake website](https://docs.delta.io/latest/delta-apidoc.html#delta-spark). ^[delta-lake-api-reference-databricks-on-aws.md]

## Related Databricks Documentation

To learn how to use the Delta Lake APIs on Databricks, refer to the following Databricks documentation:

- [What is Delta Lake in Databricks?](https://docs.databricks.com/aws/en/delta/)
- [Tutorial: Create and manage Delta Lake tables](https://docs.databricks.com/aws/en/delta/tutorial)

These resources provide guidance on working with Delta Lake tables, including creating, managing, and querying data. ^[delta-lake-api-reference-databricks-on-aws.md]
