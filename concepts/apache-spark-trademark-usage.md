# Apache Spark Trademark Usage

**Apache Spark Trademark Usage** refers to the rules and guidelines governing how the Apache Spark name, logo, and related marks may be used by third parties. Apache Spark is a project of the [Apache Software Foundation (ASF)](/concepts/apache-software-foundation-trademark-attribution.md), which maintains a formal trademark policy to protect the integrity of its project brands.

## Overview

The Apache Software Foundation holds the trademarks for Apache Spark, Spark, and the Spark logo. These marks are licensed for use under specific conditions designed to prevent confusion about the source or endorsement of products and services. ^[applicable-model-terms-databricks-on-aws.md]

## General Usage Rules

When using Apache Spark trademarks, you must follow the ASF's trademark policy. Key requirements include:

- **Attribution:** Any use of Apache Spark trademarks must include an attribution statement acknowledging the Apache Software Foundation as the trademark owner.
- **No endorsement:** You may not use the marks in a way that suggests the ASF endorses your product, service, or organization.
- **No confusion:** The marks must not be used in a manner that could cause confusion about whether a product or service is an official Apache Spark release.

## Proper Attribution

The standard attribution for Apache Spark is:

> Apache, Apache Spark, Spark and the Spark logo are trademarks of the Apache Software Foundation.

This attribution appears on Databricks documentation and other authorized materials. ^[applicable-model-terms-databricks-on-aws.md]

## Permitted Uses

You may use Apache Spark trademarks to:

- Refer to the Apache Spark project itself in a factual, descriptive manner
- Indicate that your product or service is compatible with or built on Apache Spark
- Describe the technology stack used in your documentation or marketing materials

## Prohibited Uses

You may not:

- Use Apache Spark as part of your product or company name without explicit permission
- Modify or alter the Spark logo
- Use the marks in a way that implies official status or certification
- Register domain names that include "apache-spark" or "spark" in a confusing manner

## Related Concepts

- Apache Software Foundation — The organization that governs Apache Spark and its trademarks
- Apache Spark — The open-source unified analytics engine
- Databricks — A company that builds products on Apache Spark and follows ASF trademark guidelines
- Open Source Licensing — The broader legal framework for Apache-licensed projects
